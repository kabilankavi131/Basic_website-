C++ programming



C++ programming is the second language you've to learn because in C programming you can learn only the functional programming but in C++ you can learn OOP(Object Oriented Programming) concepts.The four pillars of Object Oriented programming is :
  1. Inheritence
  2. Polymorphism
  3. Abstraction
  4. Encapsulation
You'll learn more about the above topics when you started to learn C++
